IPL Data Analysis Dashboard (2008–2025)

**Project Overview**

This project presents an end-to-end analysis of the Indian Premier League (IPL) dataset (2008–2025) using Python and Power BI.
The objective is to transform raw cricket data into meaningful business insights through data cleaning, exploratory data analysis, and interactive dashboards.

**Objectives**

- Clean and preprocess raw IPL data
- Perform exploratory data analysis (EDA)
- Build an interactive Power BI dashboard
- Analyze team, player, and venue performance
- Generate actionable insights

**Tech Stack**

- Python
- Pandas
- NumPy
- Power BI
- Power Query
- DAX
- Excel
- Jupyter Notebook

**Project Workflow**

   Raw Dataset
 
   ↓
 
   Python Data Cleaning
 
   ↓
 
   Exploratory Data Analysis
 
   ↓
 
   Export Clean Dataset
 
   ↓
 
   Power BI Dashboard
 
   ↓
 
   Business Insights

**Dashboard Pages**
* Overview
* Team Analysis
* Player Analysis
* Venue Analysis
* Insights Dashboard


**Key Insights**

- Most successful IPL teams identified.
- Top-performing batters and bowlers analyzed.
- Venue-wise scoring trends explored.
- Toss decisions compared with match outcomes.
- Seasonal performance trends visualized.
